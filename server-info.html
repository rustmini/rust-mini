Rust Mini Server Information

Rust Mini is a monthly wipe 'vanilla-ish' Rust server designed for players who work, have families, or simply prefer a more relaxed progression pace.

To prevent massive horse poo farms with players sitting afk in their bases, you will need to buy your horses, you will not find them in the wild. The horses also poo less but enough to use the fertilizer as intended.

Server Settings

  • Monthly Map Wipes
  • Vanilla Gather Rates
  • Active Non-Playing Admins
  • No Pay-to-Win Features
  • Long-Term Base Building
  • Community Focused
